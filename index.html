搭建个人博客全记录

——从零开始构建技术博客，踩坑与解决方案全复盘

一、背景与目标

作为一名学生，我希望搭建一个个人博客网站，用于记录学习笔记、技术实践和项目经验，增强求职竞争力。经过调研，最终选择 Hexo静态生成器 + GitHub Pages 的方案，兼顾技术展示与零成本托管需求。

二、技术选型与方案对比

方案 优点 缺点 适用场景
Hexo + GitHub Pages 免费、速度快、支持Markdown 需掌握Git和命令行 技术博客、开发者展示
WordPress 功能丰富、可视化操作 需服务器、成本较高 非技术用户或企业站点
Notion托管 零代码、极简操作 功能受限、SEO弱 轻量内容分享

最终选择理由：

  1. 零服务器成本，利用GitHub Pages免费托管
  2. 通过Git管理代码，增强技术能力展示
  3. 支持高度定制化主题，满足个性化需求
